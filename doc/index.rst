Welcome to ESP-IDF-TLV493D's documentation!
===========================================

Here you can find a basic documentation for the esp-idf-tlv493d library. 

This library is for the tlv493d modul to read out triple-axis magnetometer.
It is build for esp-idf framework and written in C++!

.. note:: 
   At the moment this library is in BETA phase. Future changes on api will be possible.
